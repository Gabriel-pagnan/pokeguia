
# Guia sobre pokémons 

Uma ferramenta que busca pekémons, busca feita por nome, número ou por tipo.
Criado como projeto de estudo.


## Funcionalidades

- Lista dos pokémons
- Tipo e abilidade
- Pontuação de ataques
- Pontuação de defesa
- Pontuação de velocidade
